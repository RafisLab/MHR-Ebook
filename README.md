
# সমসাময়িক সমাজতাত্ত্বিক তত্ত্ব - ই-বুক অ্যাপ

এটি একটি পূর্ণাঙ্গ অফলাইন-ফার্স্ট বাংলা ই-বুক অ্যাপ্লিকেশন যা সমাজবিজ্ঞানের শিক্ষার্থীদের জন্য তৈরি।

## বৈশিষ্ট্যসমূহ:
- ৭টি অধ্যায় ভিত্তিক বিন্যাস।
- সংক্ষিপ্ত ও রচনামূলক প্রশ্নের আলাদা ট্যাব।
- রিডার মোড (বুকমার্ক, ফন্ট সাইজ কন্ট্রোল, টেক্সট-টু-স্পিচ)।
- অফলাইন সাপোর্ট (লোকাল স্টোরেজ)।
- পাসওয়ার্ড প্রটেক্টেড শক্তিশালী এডমিন প্যানেল।
- রিচ টেক্সট এডিটর (WYSIWYG)।
- ডেটা এক্সপোর্ট/ইমপোর্ট সুবিধা।

## ভার্সেল (Vercel) ডিপ্লয়মেন্ট গাইড:

১. এই কোডটি একটি গিটহাব রিপোজিটরিতে (GitHub Repository) পুশ করুন।
২. [Vercel Dashboard](https://vercel.com/dashboard) এ যান।
